# Diary

## 프로젝트 개요

### 프로젝트 소개

하루 일기를 기록한 후, 오늘의 감정을 분석해줍니다.  
분석한 감정을 기반으로 영화, 글귀, 여행지(예정) 을 추천해 줍니다.  
친구 기능을 통해 공유일기를 작성할 수 있으며, 여러명과 함께 일기를 공유할 수 있습니다.
